About This Animation

The original sprite sheet has about 34 animations if I counted correctly (some are still frames). I've animated 8 of them corresponding to different directions of the cat sprite walking. You can use the WASD keys, including combinations, to view the different animations as the cat walks across the canvas.

About This Refactor

This project builds on top of the previous animation demo project as a refactor of its code into TypeScript. The code powering the page has been transpiled into JavaScript from the TypeScript source. Using TypeScript in development adds types, allowing code to be more reliable with fewer potential bugs. Further, the logic of this animation has been re-organized into components, providing flexibility when used as a base for larger projects. I originally stumbled over something called the Entity Component System (ECS), which would have gone even further and organized the game completely out of components with entities containing almost nothing but an ID. Ultimatly I decided not to go quite that far. Instead, entities are instantiated and behavior components added to them, defining their behavior.
